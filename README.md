Программа на C++ находит числа Фибоначчи в заданном диапазоне.

Генерируютсч числа Фибоначчи с помощью цикла, начиная с 0 и 1, а каждое следующее число — это сумма двух предыдущих.

Программа проверяет, попадает ли число в нужный диапазон или выдает сообщение о том, что в этом диапазоне нету чисел Фибоначчи.

Я продумал возможные ошибки ввода.  
Например, если первое число больше или равно второму, то выводится сообщение об ошибке.

Также я добавил проверку, чтобы пользователь вводил именно числа.  
Если ввести буквы или символы, то выводится сообщение о том, что надо вводить числа.

Я проверял программу на сайте OnlineGDB и тестировал разные диапазоны и ввод с ошибкой.
